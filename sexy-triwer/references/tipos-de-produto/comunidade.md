# Comunidade

| | |
|---|---|
| **Exemplo real** | O Novo Mercado |
| **Entrega** | Área de membros |
| **Ticket comum** | R$79/mês – R$2.997/ano |
| **Vantagem** | Duas formas de construir: várias aulas semanais (clássico) ou networking (mais trabalhoso) |
| **Desvantagem** | Modelo mais longo prazo que existe no mercado |
| **Posição no funil** | Produto principal e/ou upsell |

A forma "aulas semanais" exige constância, bagagem e persistência — em
geral mais barato, funciona como renda passiva a longo prazo. A forma
"networking" é mais influente, exige posicionamento à prova de falhas e
boa plataforma — costuma começar como curso e ir migrando aos poucos.

**Perguntas para estruturar:**
1. Eu quero que meu produto seja maior que eu?
2. Estou disposto a esperar anos pelo resultado?
3. Sou uma pessoa influente com propósito de montar comunidade, ou apenas
   de ensinar?

**Alerta:** foco extremo no longo prazo, a não ser que seja nicho de oceano
azul. Difícil sobreviver em modalidade assinatura. Não recomendado para
negócios sem estrutura ainda.

**Pilares dominantes:** Inveja, Vaidade (Luxúria), Ira, Preguiça e Gula —
ver `../pilares-do-desejo/`. Este é um formato que **depende** do
pertencimento/autoimagem do grupo — por isso o ticket precisa ficar acima
de R$797/ano (ver regra estrutural em `INDEX.md`).
