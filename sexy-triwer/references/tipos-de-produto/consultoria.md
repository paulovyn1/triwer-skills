# Consultoria

| | |
|---|---|
| **Entrega** | WhatsApp e Zoom |
| **Ticket comum** | R$697+ (céu é o limite) |
| **Vantagem** | É pontual, depois do problema resolvido tem um fim |
| **Desvantagem** | Ocupa muitas horas da agenda |
| **Posição no funil** | Cross sell, downsell ou upsell |

A diferença principal entre consultoria e mentoria não é o tempo (uma
consultoria pode durar mais que uma mentoria) — é que a consultoria propõe
**resolver um problema ultra-específico** (comum no mundo corporativo,
papel de encontrar soluções), enquanto a mentoria propõe **acompanhar até
um resultado desejado**.

**Perguntas para estruturar:**
1. Quais problemas sou capaz de resolver?
2. Como posso processualizar essas soluções?

**Alerta:** pode ser um tiro no pé sem um bom método por trás.

**Pilar dominante:** Segurança (fora dos 7 pilares — ver observação em
`../pilares-do-desejo/INDEX.md`). Também pode usar Avareza quando ancorado
contra o custo de resolver o problema sozinho ou de forma errada.
