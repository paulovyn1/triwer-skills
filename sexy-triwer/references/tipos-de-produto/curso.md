# Curso

| | |
|---|---|
| **Exemplo real** | Vendas Automáticas |
| **Entrega** | Área de membros |
| **Ticket comum** | R$197–997 |
| **Vantagem** | Produto clássico e "simples" de estruturar |
| **Desvantagem** | Mais complexo de vender sem foco específico |
| **Posição no funil** | Produto principal, upsell e downsell |

O que separa um curso de uma formação é a quantidade de aulas e a
especificidade — um curso tem um propósito, mas não promete tornar a
pessoa uma especialista (isso é papel da Formação).

**Perguntas para estruturar:**
1. De que ponto (a) até que ponto (b) posso levar minha audiência?
2. O que é imprescindível para que alcancem o resultado prometido?

**Alerta — o maior perigo deste formato:** aulas teóricas demais e
desnecessárias entediam o lead e tornam o curso "inútil". Colocar coisa
demais dentro do curso faz ele perder o objetivo e virar apenas uma grande
biblioteca de conhecimento.

**Pilares dominantes:** Preguiça, Ira e Gula — ver `../pilares-do-desejo/`.
