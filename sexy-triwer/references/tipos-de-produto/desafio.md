# Desafio

| | |
|---|---|
| **Exemplo real** | Perpétuo Paquera, Malha de Ouro |
| **Entrega** | WhatsApp, lives ao vivo e/ou área de membros |
| **Ticket comum** | R$97–497 |
| **Vantagem** | Ótimo produto de lançamento, engaja muito a audiência, gera muito depoimento |
| **Desvantagem** | Difícil de colocar no perpétuo, depende muito de interação |
| **Posição no funil** | Produto de intermediário e/ou cross sell |

Perfeito para validar uma metodologia ou construir comunidade quando o
perfil ainda é pequeno. Gera muito depoimento para impulsionar vendas
depois.

**Perguntas para estruturar:**
1. De que ponto (a) até que ponto (b) posso levar minha audiência?
2. Como eu organizaria isso passo a passo com tarefas?

**Alerta:** produto de difícil escala porque o que move as pessoas é o
"evento" — mas pode ser incorporado em grupos de mentoria, principalmente
em nichos como área da saúde.

**Nota — Malha de Ouro:** este tipo está catalogado como o formato do
Malha de Ouro, mas na prática ele é um híbrido Desafio+Pacote — ver
`INDEX.md` para o raciocínio completo. Não assuma que todo produto
rotulado "Desafio" vende pelo mesmo pilar (Agilidade) — confirme o pilar
dominante separadamente na Etapa 3a.
