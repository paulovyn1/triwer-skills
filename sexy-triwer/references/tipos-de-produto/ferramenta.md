# Ferramenta

| | |
|---|---|
| **Exemplo real** | Conteúdo para Lançamentos |
| **Entrega** | Trello, Notion ou PDF — foco 100% prático |
| **Ticket comum** | R$97–297 |
| **Vantagem** | Ótimo produto perpétuo, vende muito facilmente |
| **Desvantagem** | Lucro menor, depende de tráfego direto |
| **Posição no funil** | Produto de entrada e/ou orderbump |

Visa **facilitar** a vida do lead num tema muito específico — não ensina,
resolve. Ótimo para tornar outra oferta mais irresistível também (pode ser
incorporado como bônus dentro de outro produto).

**Perguntas para estruturar:**
1. Eu usaria isso no meu dia a dia?
2. Que problema esse produto resolve?
3. Como ele economiza o tempo do meu público?
4. Como eu posso deixá-lo mais visual?

**Alerta:** se não for específico, dificilmente vende no tráfego pago.

**Pilares dominantes:** Preguiça (o mais importante aqui) e Gula — ver
`../pilares-do-desejo/`. Segurança (fora dos 7 pilares) também aparece
quando o produto reduz o medo de "fazer errado".
