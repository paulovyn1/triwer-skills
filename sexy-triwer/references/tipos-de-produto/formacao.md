# Formação

| | |
|---|---|
| **Exemplo real** | Sim Inevitável |
| **Entrega** | Área de membros |
| **Ticket comum** | R$997–2.997 |
| **Vantagem** | Perfeito para lançamentos, tende a vender mais fácil |
| **Desvantagem** | Pouco senso de comunidade, foco maior sempre na autoridade |
| **Posição no funil** | Produto principal e/ou upsell |

Viés mais "profissionalizante" que Comunidade — o foco não é networking, é
a habilidade que a pessoa vai adquirir (um fator externo: virar gestor de
tráfego bem pago, co-produtor requisitado, dominar uma habilidade
específica). Pode evoluir para Comunidade com o tempo, caminho mais
escalável.

**Perguntas para estruturar:**
1. Qual a habilidade/profissão primária que preciso ensinar?
2. Eu tenho um método validado para isso?
3. Qual o caminho mais curto até lá?

**Alerta:** produto para ser construído no dia a dia, não de uma vez.

**Pilares dominantes:** Inveja, Vaidade (Luxúria), Ira, Preguiça e Gula —
ver `../pilares-do-desejo/`. Formato que depende do pertencimento/
autoimagem do grupo — ticket precisa ficar acima de R$797/ano (ver regra
estrutural em `INDEX.md`).
