# Pacotes

| | |
|---|---|
| **Exemplo real** | Modelos de páginas (PG Ouro), Pack Canva |
| **Entrega** | Arquivos ou Trello |
| **Ticket comum** | R$97–197 |
| **Vantagem** | Ótimo produto perpétuo, vende muito facilmente |
| **Desvantagem** | Reembolso alto, cliente pode não andar na esteira |
| **Posição no funil** | Produto de entrada e/ou orderbump |

Parecido com Ferramenta, mas o propósito é diferente: o objetivo é
fornecer **materiais** em quantidade que ajudem numa situação específica —
não facilitar um processo, e sim entregar volume validado pronto pra usar.

**Perguntas para estruturar:**
1. Tem algo que eu use com frequência e possa disponibilizar (filtros,
   actions, texturas, modelos)?
2. Isso seria do desejo de outras pessoas?

**Alerta:** cuidado com propriedade intelectual do material empacotado.

**Pilares dominantes:** Gula (volume validado) e Avareza (percepção de
vantagem no preço) — ver `../pilares-do-desejo/`. Preguiça também aparece,
mas em segundo plano: o valor central aqui não é "facilitar um processo",
é "ter muito material pronto e confiável".
